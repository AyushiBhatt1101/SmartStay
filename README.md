# SmartStay

SmartStay is a web application designed to manage homestays efficiently. It provides features for booking management, guest feedback, and review analytics.

## Features

- **Booking Management**: Easily manage bookings for your homestay.
- **Guest Feedback**: Collect and analyze feedback from guests to improve services.
- **Review Analytics**: Gain insights into guest reviews and ratings.

## Pages

- **Home**: The landing page featuring a hero section, features, about section, and footer.
- **Login**: A page for users to log in to their accounts.
- **Register**: A page for new users to create an account.
- **Listings**: A page displaying a list of available homestays.
- **Details**: A page showing detailed information about a specific homestay.
- **Dashboard**: A user dashboard displaying key metrics and statistics.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/smartstay.git
   ```

2. Navigate to the project directory:
   ```
   cd smartstay
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

## Tech Stack

- **Frontend**: React
- **Styling**: Tailwind CSS
